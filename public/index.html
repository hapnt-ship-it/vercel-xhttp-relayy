🚀 XHTTP Relay

Vercel Edge Function for Xray/V2Ray XHTTP Transport

Service Status: Online
The relay is operational and ready to forward XHTTP traffic.

About This Service

This is a minimal Vercel Edge Function that relays XHTTP traffic to a backend Xray/V2Ray server. It leverages Vercel's globally distributed edge network to provide low-latency access.

Features

  • True bidirectional streaming via WebStreams
  • Runs on Vercel's global edge network
  • ~5-50ms cold start times
  • XHTTP transport support only
  • Automatic header filtering and forwarding

Configuration

This relay forwards requests to the configured TARGET_DOMAIN environment variable. For setup instructions and client configuration examples, please refer to the GitHub repository.
